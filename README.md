# 📊 Telco Customer Churn Analysis

## 📌 Overview
This project analyzes the **Telco Customer Churn Dataset** using **Python, Pandas, Matplotlib, and Seaborn** to identify the key factors influencing customer churn. Through Exploratory Data Analysis (EDA), the project uncovers patterns in customer demographics, contract types, payment methods, internet services, and value-added services to generate actionable business insights for improving customer retention.

## 🎯 Objectives
- Perform data cleaning and preprocessing.
- Analyze customer churn patterns using visualizations.
- Identify factors contributing to customer attrition.
- Provide business recommendations to reduce churn.

## 🛠️ Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📈 Key Findings
- **26.54%** of customers churned, while **73.46%** were retained.
- **55.02%** of customers are on month-to-month contracts, making them the highest-risk segment.
- **Fiber Optic** customers have the highest churn rate (**41.89%**).
- Customers without **Online Security** (**41.77%**) and **Tech Support** (**41.64%**) are significantly more likely to churn.
- Automatic payment methods are associated with better customer retention than Electronic Check.

## 💡 Business Recommendations
- Promote long-term contracts.
- Improve the Fiber Optic customer experience.
- Bundle Online Security and Tech Support with internet plans.
- Encourage automatic payment methods.
- Target new customers with personalized retention offers.
